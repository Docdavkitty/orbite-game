# 🌀 Orbite — Jeu de réflexes orbital

Survivez le plus longtemps possible en orbite autour d'un soleil, en évitant les astéroïdes.

**[🎮 Jouer en ligne](https://docdavkitty.github.io/orbite-game/)**

## Comment jouer

- **↑ / ↓** — Changer d'orbite (intérieur / extérieur)
- **ESPACE** — Dash latéral (permet d'esquiver rapidement, cooldown ~2s)
- **R** — Recommencer

Sur mobile : swipe vertical pour changer d'orbite, tap pour dash.

## Règles

- Vous orbitez autour d'un soleil sur 3 couronnes concentriques
- Des astéroïdes arrivent de toutes les directions
- Changez d'orbite ou dash pour les éviter
- La vitesse et la densité augmentent avec le niveau (tous les 300 points)
- Record sauvegardé dans le navigateur

## Déploiement

Jeu 100% statique (HTML/CSS/JS vanilla). Ouvrez `index.html` dans un navigateur ou hébergez-le sur GitHub Pages / Netlify / Vercel — zéro dépendance.
